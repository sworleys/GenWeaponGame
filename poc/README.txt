---------------------------
GENETIC ALGORITHM PROTOTYPE
---------------------------
   
This is the prototype of the genetic algorithm we planned to implement in Godot. 

--------
CONTROLS
--------
Player 1 uses keys 1-4 on the number row to select their weapon.
Player 2 uses keys 7-8 on the number row to select their weapon.

LEFT MOUSE CLICK to see the weapons fire.

The ENTER key will simulate the round ending and will have the genetic algorithm run.
